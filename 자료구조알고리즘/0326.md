- 추상적 자료구조
  - Data : 정수, 문자열, 레코드 등
  - A set of operations : 삽입, 삭제, 순회, 정렬, 탐색 등

- 연결리스트
  - Node로 이루어져있고 Node는 데이터와 링크로 연결
  - 선형 탐색과 유사 O(n)
  - 연산 정의
    - 길이 얻기
    - 리스트 순회
    - 특정 원소 참조
    - 원소 삽입
      - 포지션이 아니라 특정 노드에서 삽입하면 연산 절약 가능
      - 더미 노드를 활용해 편하게 사용
    - 원소 삭제
    - 두 리스트 합치기
  - 장점
    - 삽입과 삭제가 유연하다
